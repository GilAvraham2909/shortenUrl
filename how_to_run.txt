1. Go into the shortenUrl folder.
2. Install node on your computer:
	for ubuntu users: sudo apt install nodejs
3. Install dependencies:
	run in terminal: npm install express dotenv
4. Run the Express server:
	run in terminal: node server.ts
5. Now you're shorten URL service is up and running on localhost!
	you can use the following endpoints:
	A. To get short url for your url do: /encode?url=[url_to_be_shorten]
	B. To get the long URL you have shorten do: /encode?url=[given_shorten_url]
